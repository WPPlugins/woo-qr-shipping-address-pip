== Description ==

Adds a QR Code with the Shipping Address link on Google Maps into "Print Invoices & Packing lists" (woocommerce.com) and "WooCommerce Print Invoice & Delivery Note" (wordpress.org) plugins.

Useful for setting the navigation destination address directly in the Google Maps app on your smartphone when you're delivering your orders.

Uses the [QR code API](http://goqr.me/api/) on goqr.me

= Features: =

* Inserts a QR Code with the Shipping Address link on Google Maps, that can be scanned by the delivery courier with his smartphone, into documents generated by the following plugins:
	* [Print Invoices & Packing lists](https://woocommerce.com/products/print-invoices-packing-lists/)
	* [WooCommerce Print Invoice & Delivery Note](https://wordpress.org/plugins/woocommerce-delivery-notes/)
	* If you need to insert the QR Code on any other plugin document or admin screen, [drop us a line](https://www.webdados.pt/contactos/).
* WPML tested and compatible;
* PHP7 tested and compatible;

== Installation ==

* Use the included automatic install feature on your WordPress admin panel and search for “QR Shipping Address WooCommerce PIP”.
